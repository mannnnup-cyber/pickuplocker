# Introduction

This pilot project tests the viability of \'Pickup\' --- a smart locker
solution jointly explored by Dirty Hand Designs (DHD) and 876OnTheGo.
The locker is already installed at the UTech campus. The goal is to test
smart, contactless order pickups, explore future integration potential,
and validate its use as a commercial service.

# Branding Strategy

The locker frame is complete. Branding will include a vinyl or ACM wrap
with Pickup logo, instructional graphics, and a QR code help system. The
front panel will showcase benefits like 24/7 pickup, online ordering,
and automated access.

# Manual Operations Plan Until System Launch

During the pilot phase, until the automated locker dashboard is built,
the Pickup locker will operate as follows:

-   DHD and 876OnTheGo will manually assign locker compartments upon
    confirming payment.

-   Customers will receive a manual message (WhatsApp or SMS) with
    pickup code and location.

-   Each assigned pickup will be tracked using a shared Google Sheet or
    Airtable log.

-   Unclaimed items after 3 days will be flagged, and customers notified
    of a storage fee.

-   DHD and 876OnTheGo will manage locker loading and code distribution
    collaboratively.

# Support Protocol (Offline Contingency)

If the locker goes offline, customers will not be able to retrieve
items. A Doorvi-powered QR will link customers to a DHD or 876 support
contact. A GSM line will serve as backup. A master key will allow manual
retrieval. Instructions will be posted on the locker.

# Conclusion

services, and partner locations across Jamaica.
